# calendar
A C program to show calendar (of AD and BS system) and allow date conversion

# Build, Run 
## Linux 
```
make all
make run
```
 
 
## Windows 
* Open the Code Blocks Project file .cbp
* Ctrl+F9 (Build and Run)
